使用教程

详细安装指南

从下载到上手，每一步都有清晰指引

1

下载代码

访问 GitHub 或 Gitee 仓库，点击页面上的绿色 「Code」 按钮，选择 「Download ZIP」 下载压缩包。

下载完成后，右键解压 ZIP 文件到任意目录（例如桌面或 D 盘），记住解压后的文件夹位置。

2

打开扩展管理页面

打开 Chrome 浏览器，在地址栏输入 chrome://extensions 并回车。

在页面右上角，找到并打开 「开发者模式」 开关。

3

加载扩展程序

点击左上角出现的 「加载已解压的扩展程序」 按钮。

在弹出的文件选择窗口中，找到并 选中第 1 步解压出来的文件夹（文件夹内应包含 manifest.json 文件），点击「选择文件夹」。

加载成功后，扩展列表中会出现 「BOSS直聘助手」 卡片。

4

配置 API Key

点击 Chrome 右上角 拼图图标 🧩 → 找到「BOSS直聘助手」→ 点击📍图钉固定到工具栏。

点击工具栏中的 ✦ 扩展图标 → 点击 「⚙️ 设置」按钮。

在设置页面中：

  • 填写个人信息（姓名、技能、经验、期望薪资）
  • 选择 AI 模型：DeepSeek（推荐，快速稳定）或 MiMo 2.5（推理模型，质量更高）
  • 填入对应模型的 API Key（DeepSeek 获取地址）
  • （可选）上传 PDF 简历，AI 自动提取信息用于生成更精准的回复

点击 「💾 保存设置」。

5

开始使用 🎉

打开 BOSS直聘（zhipin.com）或 智联招聘（zhaopin.com），即可体验：

  • 职位详情页 → 右侧自动弹出 AI 对话面板，点击「👋 打招呼」一键生成招呼语
  • 聊天页面 → 粘贴 HR 消息，点击「🤖 发送给AI」自动生成回复
  • 职位列表页 → 已投递的职位会自动显示「✓ 已沟通」绿色徽章
  • 点击扩展图标 → 查看今日投递统计、回复率等数据
FAQ

常见问题

支持哪些招聘平台？−
支持 BOSS直聘（boss.cn / zhipin.com）和 智联招聘（zhaopin.com / zhaopin.cn）。选择器系统支持多级回退，网站改版时大部分功能可自愈。
MiMo 和 DeepSeek 如何选择？+
DeepSeek：响应快速，稳定性好，推荐日常使用。
MiMo 2.5：推理模型，回复质量更高但耗时较长（45s超时）。扩展已对 MiMo 的 reasoning_content 字段做兼容处理。
我的数据安全吗？+
绝对安全。所有数据存储在浏览器本地（IndexedDB + chrome.storage.local），不上传任何服务器。AI API 调用仅发送当前对话上下文。
为什么点击按钮没有反应？+
请检查：1) 是否已配置 API Key；2) 当前页面是否为职位详情页或聊天页；3) 点击右下角 🐛 调试按钮查看日志；4) F12 控制台搜索 [BossHelper] 确认扩展已加载。
